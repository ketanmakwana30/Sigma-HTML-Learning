## HTML Notes

### 1. What is HTML?
HTML (HyperText Markup Language) is used to structure a webpage.


### 3. Important Topics

- **Headings** – Webpage me title ya sub-title dene ke liye (`<h1>` to `<h6>`).
- **Paragraphs** – Normal text likhne ke liye (`<p>` tag).
- **Links** – Kisi page ya website par navigate karne ke liye (`<a href="">`).
- **Images** – Webpage me images display karne ke liye (`<img src="" alt="">`).
- **Lists** – Ordered (`<ol>`), Unordered (`<ul>`), and Description lists (`<dl>`).
- **Forms** – User se input lene ke liye (`<form>`, `<input>`, `<button>`).
- **Tables** – Data ko rows & columns me show karne ke liye (`<table>`, `<tr>`, `<td>`).
- **Semantic Tags** – SEO & structure improve karne ke liye (`<header>`, `<nav>`, `<section>`, `<footer>`).
- **Attributes** – Tags me additional info dene ke liye (id, class, src, href).
- **Block & Inline Elements** – Block full width lete hain; inline sirf content ke according.
- **Comments** – Code me notes likhne ke liye (`<!-- comment -->`).
- **Meta Tags** – SEO, character-set & responsiveness ke liye (`<meta>`).
- **HTML Entities** – Special symbols show karne ke liye (`&lt;`, `&gt;`, `&copy;`).
